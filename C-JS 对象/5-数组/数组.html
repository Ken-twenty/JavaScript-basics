数组对象的作用是：使用单独的变量名来存储一系列的值。

Array对象的方法

  1. new 创建方法

    • var cars = new Array(); cars[0]="Saab"; cars[1]="Volvo"; cars[2]="BMW";

    • var cars=new Array(3); cars[0]="Saab"; cars[1]="Volvo"; cars[2]="BMW";

    • var cars = new Array("Saab","Volvo","BMW");

  2. 访问数组

    array[index];

  3. 修改已有数组中的值

    array[index] = newValue
