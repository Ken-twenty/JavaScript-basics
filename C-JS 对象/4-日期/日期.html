日期对象用于处理日期和时间。

JavaScript Date（日期）对象 方法

  1. Date()

                document.write(Date()); // Fri Jul 07 2017 15:37:20 GMT+0800 (中国标准时间)
    返回当日的日期时间对象

  2. getTime()

                var date = new Date();
            document.write(date.getTime()); // 1499413381421
    返回从 1970 年 1 月 1 日至今的毫秒数。

  3. setFullYear()

                var date = new Date();
            date.setFullYear(2017, 01, 05);
            document.write(date); // Sun Feb 05 2017 15:55:34 GMT+0800 (中国标准时间)
    year参数必要，month、day参数可选。 月份从0开始。 三个参数都可以是任意数字，小数将向下取整，结果将按实际情况处理。

  4. toUTCString()

                var date = new Date();
            document.write(date.toUTCString()); // "Fri, 07 Jul 2017 08:14:04 GMT"
    UTC，协调世界时；GMT，格林尼治时间。在不精确到秒的情况下，视二者等同。 返回一个字符串。

  5. getDay()

                var date = new Date();
            document.write(date.getDay()); // 5（周五）
    返回一个数字 0-6，周日-周六

  6. getHours()、getMinutes()、getSeconds()

                var date = new Date();
            console.log(date.getHours(), date.getMinutes(), date.getSeconds()); // 17 2 5
    返回数字 补0
