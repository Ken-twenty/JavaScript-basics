Math 对象

  1. Math（算数）对象的作用是：执行普通的算数任务。
  2. Math 对象提供多种算数值和函数。无需在使用这个对象之前对它进行定义。

算数值

  • 自然常数：Math.E // (1+1/n)^n
  • 圆周率：Math.PI
  • 2 的平方根：Math.SQRT2
  • 1/2 的平方根：Math.SQRT1_2
  • 2 的自然对数：Math.LN2 // ex = 2
  • 10 的自然对数：Math.LN10 // ex = 10
  • 以 2 为底的 e 的对数：Math.LOG2E // 2x = e
  • 以 10 为底的 e 的对数：Math.LOG10E // 10x = e

函数（方法）

  1. round()

                document.write(Math.round(0.60))  // 1
            document.write(Math.round(0.50))  // 1
            document.write(Math.round(0.49))  // 0
            document.write(Math.round(-4.40)) // -4
            document.write(Math.round(-4.60)) // -5
    必要参数、必须是数字。返回四舍五入的整数

  2. random()

                document.write(Math.random()) // 7.1234567890123456 （小数位上限为17位）
    使用 random() 来返回 0 到 1 之间的随机数。

  3. max()

                document.write(Math.max(5,7))       // 7
            document.write(Math.max(-3,5))      // 5
            document.write(Math.max(-3,-5))     // -3
            document.write(Math.max(7.25,7.30)) // 7.3
    返回若干个参数中最大的值。 如果没有参数，则返回 -Infinity。 如果有某个参数为 NaN，或是不能转换成数字的非数字值，则返回 NaN。

  4. min()

                document.write(Math.min(5,7))       // 5
            document.write(Math.min(-3,5))      // -3
            document.write(Math.min(-3,-5))     // -5
            document.write(Math.min(7.25,7.30)) // 7.25
    返回若干个参数中最小的值。 如果没有参数，则返回 Infinity。 如果有某个参数为 NaN，或是不能转换成数字的非数字值，则返回 NaN。
