String 对象用于处理已有的字符块。

JavaScript String（字符串）对象 实例

  1. 计算字符串的长度

                var txt = 'Talk is cheap.'
            console.log(txt.length);

  2. 为字符串添加样式（不推荐，应该实现关注点分离）

                var txt="Show me the code.";

            txt.big(); // "<big>Show me the code.</big>"

            txt.small(); // "<small>Show me the code.</small>"

            txt.bold(); // "<b>Show me the code.</b>"

            txt.italics(); // "<i>Show me the code.</i>"

            txt.blink(); // "<blink>Show me the code.</blink>" （闪烁，大部分浏览器不支持）

            txt.fixed(); // "<tt>Show me the code.</tt>" （等宽文本）

            txt.strike(); // "<strike>Show me the code.</strike>" （删除线）

            txt.fontcolor(); // "<font color="#abcdef">Show me the code.</font>"

            txt.fontsize(20); // "<font size="20">Show me the code.</font>"

            txt.toLowerCase(); // "show me the code."

            txt.toUpperCase(); // "SHOW ME THE CODE."

            txt.sub(); // "<sub>Show me the code.</sub>" （下标）

            txt.sup(); // "<sup>Show me the code.</sup>" （上标）

            txt.link("http://www.w3school.com.cn"); // "<a href="http://www.w3school.com.cn">Show me the code.</a>"
可以关注下sub、sup标签 返回结果都是字符串，可用document.write()输出。

indexOf() 方法

  1. 使用 indexOf() 来定位字符串中某一个指定的字符首次出现的位置。
  2. 空格也算入字符串长度。
  3. 返回结果为-1时，不存在。

match() 方法

  1. 使用 match() 来查找字符串中特定的字符，并且如果找到的话，则返回这个字符。
  2. 返回结果为null时，不存在

replace() 方法

  1. 使用 replace(匹配对象, 替换文本) 方法在字符串中用某些字符替换另一些字符。
  2. 匹配对象可以是RegExp或者String
  3. 只替换匹配到的第一个（段）文本，匹配不成功则返回原字符串
