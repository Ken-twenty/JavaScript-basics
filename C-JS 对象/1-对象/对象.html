JavaScript 中的所有事物都是对象：字符串、数值、数组、函数...

此外，JavaScript 允许自定义对象。

JavaScript 对象

  1. JavaScript 提供多个内建对象，比如 String、Date、Array 等等。
  2. 对象只是带有属性和方法的特殊数据类型。

访问对象的属性

  1. 属性是与对象相关的值。访问对象属性的语法是：
    objectName.propertyName

访问对象的方法

  1. 方法是能够在对象上执行的动作。您可以通过以下语法来调用方法：
    objectName.methodName()

创建 JavaScript 对象

  1. 通过 JavaScript，您能够定义并创建自己的对象。创建新对象有两种不同的方法：
    1. 定义并创建对象的实例 【new Object()、字面量对象】
    2. 使用函数来定义对象，然后创建新的对象实例【new 自定义构造函数()】
还有Object.create(obj)，obj作为新对象的原型

把属性添加到 JavaScript 对象

  1. 您可以通过为对象赋值，向已有对象添加新属性
  2. 假如obj已存在，你可以为其添加新属性及值：
    obj.newKey = newValue;

把方法添加到 JavaScript 对象

  1. 方法只不过是附加在对象上的函数。
  2. 在构造器函数内部定义对象的方法：
    function person(firstName,lastName,age,eyeColor) {
        this.firstName=firstName;
        this.lastName=lastName;
        this.age=age;
        this.eyeColor=eyeColor;

        this.changeName=changeName;

        function changeName(name) {
            this.lastName=name;

    }

JavaScript 类

  1. JavaScript 是面向对象的语言，但 JavaScript 不使用类。
  2. 在 JavaScript 中，不会创建类，也不会通过类来创建对象（就像在其他面向对象的语言中那样）。
  3. JavaScript 基于 prototype，而不是基于类的。

`JavaScript for...in 循环

  1. JavaScript for...in 语句循环遍历对象的属性。
    for (key in obj) {
            // 要执行的代码
            console.log(key + ':' + obj[key]);
    }
for...in 循环中的代码块将针对每个属性（key）执行一次。
