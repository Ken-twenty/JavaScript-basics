JavaScript Number 对象

  1. JavaScript 只有一种数字类型。
  2. 可以使用也可以不使用小数点来书写数字。

JavaScript 数字

  1. JavaScript 数字可以使用也可以不使用小数点来书写
  2. 极大或极小的数字可通过科学（指数）计数法来写：
    var y=123e5;    // 12300000
    var z=123e-5;   // 0.00123

所有 JavaScript 数字均为 64 位

  1. JavaScript 不是类型语言。与许多其他编程语言不同，JavaScript 不定义不同类型的数字，比如整数、短、长、浮点等等。
  2. JavaScript 中的所有数字都存储为根为 10 （指十进制数字）的 64 位（8 比特），浮点数。
任何数都可以用 n = s * 2^p 表示，即 n = 尾数 * 底数 ^ 阶码，进制决定底数；阶码决定小数点的位置。 阶码为固定值时，这样的数称为定点数；阶码可变时，这样的数称为浮点数。 十进制转二进制，很多数字的转化结果往往不能确定小数点的位置（循环小数），因而会有精度取舍。 js的展示层是十进制，底层运算是二进制。

精度

  1. 整数（不使用小数点或指数计数法）最多为 15 位。
  2. 小数的最大位数是 17，但是浮点运算并不总是 100% 准确（见上文）

八进制和十六进制

  1. 如果前缀为 0，则 JavaScript 会把数值常量解释为八进制数。
  2. 如果前缀为 0 和 "x"，则解释为十六进制数。

数字属性和方法（Number对象）

  1. 属性

    • MAX VALUE（最大值）
    • MIN VALUE（最小值）
    • NEGATIVE INFINITY（负无穷大）
    • POSITIVE INFINITY（正无穷大）
    • NaN（不是一个数字）
    • prototype（原型）
    • constructor（构造函数是Function）

  2. 方法

    • toExponential()（可把对象的值转换成指数计数法）
    • toFixed()（四舍五入，可控制小数位个数，0-20）
    • toPrecision()（同toExponential，但可控制有效数字的个数，1-21；底数第一位一定是个位）
    • toString()（把Number对象的数字转化为字符串类型）
    • valueOf()（返回Number对象的数字值）
    var numA = new Number(520);
    var resultA = numA.toExponential(); // 5.2e+2

    var numB = new Number(5.2);
    var resultB = numB.toFixed(2); // "5.20"

    var numC = new Number(520000);
    var resultC =numC.toPrecision(2); // "5.2e+5"

    var numD = new Number(520000);
    var resultD =numD.toString(2); // "520000"

    var numE = new Number(520000);
    var resultE = numE.valueOf(); // 520000
