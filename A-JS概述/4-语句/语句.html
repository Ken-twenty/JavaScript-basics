JavaScript 语句

  1. JavaScript 语句是向浏览器发出的命令。
  2. 语句的作用是告诉浏览器该做什么。

分号 ;

  1. 分号用于分隔 JavaScript 语句。
  2. 使用分号的另一用处是在一行中编写多条语句。
在 JavaScript 中，用分号来结束语句是可选的。

JavaScript 代码

  1. JavaScript 代码是 JavaScript 语句的序列。
  2. 浏览器会按照编写顺序来执行每条语句。

JavaScript 代码块

  1. 块由左花括号开始，由右花括号结束。
  2. 块的作用是使语句序列一起执行
JavaScript 函数是将语句组合在块中的典型例子。

JavaScript 对大小写敏感

  1. JavaScript 对大小写是敏感的。
  2. 当编写 JavaScript 语句时，请留意是否关闭大小写切换键。

空格

  1. JavaScript 会忽略多余的空格。
您可以向脚本添加空格，来提高其可读性。

对代码行进行折行

  1. 您可以在文本字符串中使用反斜杠对代码行进行换行。
    <script>
        document.write('代码换\
    行');
    </script>
