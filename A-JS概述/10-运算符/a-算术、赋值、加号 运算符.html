JavaScript 算术运算符

  1. 算术运算符用于执行变量与/或值之间的算术运算。
y = 5
运算符 描述 例子 结果
+ 加 x=y+2 x=7
- 减 x=y-2 x=3
* 乘 x=y*2 x=10
/ 除 x=y/2 x=2.5
% 求余数 (保留整数) x=y%2 x=1
++ 累加 x=++y x=6
-- 递减 x=--y x=4

JavaScript 赋值运算符

  1. 赋值运算符用于给 JavaScript 变量赋值。
x=10;y=5
运算符 例子 等价于 结果
= x=y x=5
+= x+=y x=x+y x=15
-= x-=y x=x-y x=5
*= x*=y x=x*y x=50
/= x/=y x=x/y x=2
%= x%=y x=x%y x=0

用于字符串的 + 运算符

  1. + 运算符用于把文本值或字符串变量加起来（连接起来）。

对字符串和数字进行加法运算

  1. 如果把数字与字符串相加，结果将成为字符串。
