比较和逻辑运算符用于测试 true 或 false。

比较运算符

  1. 比较运算符在逻辑语句中使用，以测定变量或值是否匹配。
x=5
运算符 描述 例子
== 等于 x==8 为 false
=== 全等（值和类型） x===5 为 true；x==="5" 为 false
!= 不等于 x!=8 为 true
> 大于 x>8 为 false
< 小于 x<8 为 true
>= 大于或等于 x>=8 为 false
<= 小于或等于 x<=8 为 true

逻辑运算符

  1. 逻辑运算符用于测定变量或值之间的语句逻辑。
x=6;y=3
运算符 描述 例子
&& and (x < 10 && y > 1) 为 true
|| or (x==5 || y==5) 为 false
! not !(x==y) 为 true

条件运算符（三元运算符）

  1. JavaScript 还包含了 基于某些条件 对代码语句进行选择的 条件运算符：
    variableName=(condition)?value1:value2
condition == true -> value1, condition == false -> value2
