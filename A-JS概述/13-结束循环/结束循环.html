break 语句用于跳出循环。
continue 用于跳过循环中的一个迭代。

Break 语句

  1. 我们已经在本教程稍早的章节中见到过 break 语句。它用于跳出 switch() 语句。
  2. break 语句可用于跳出循环。
  3. break 语句跳出循环后，会继续执行该循环之后的代码（如果有的话）

Continue 语句

  1. 如果出现了指定的条件，continue 语句将中断正在循环中的迭代，然后继续循环中的下一个迭代。

JavaScript 标签

  1. 正如您在 switch 语句那一章中看到的，可以对 JavaScript 语句进行标记。（case）
  2. 如需标记 JavaScript 语句，请在语句之前加上冒号：
    label:
    语句;
break 和 continue 语句仅仅是能够跳出代码块的语句。 
    break labelName;

    continue labelName;
  1. continue 语句（带有或不带标签引用）只能用在循环中。
  2. break 语句（不带标签引用），只能用在循环或 switch 中。
  3. 通过标签引用，break 语句可用于跳出任何 JavaScript 代码块
