循环可以将代码块执行指定的次数。

JavaScript 循环

  1. 如果您希望一遍又一遍地运行相同的代码，并且每次的值都不同，那么使用循环是很方便的。

不同类型的循环

  1. for - 循环代码块一定的次数
  2. for/in - 循环遍历对象的属性
  3. while - 当指定的条件为 true 时循环指定的代码块
  4. do/while - 同样当指定的条件为 true 时循环指定的代码块

for 循环

    for (语句 1; 语句 2; 语句 3) {
        被执行的代码块
    }
  1. 语句 1 在循环（代码块）开始前执行
  2. 语句 2 定义运行循环（代码块）的条件
  3. 语句 3 在循环（代码块）已被执行之后执行

语句1

  1. 通常我们会使用语句 1 初始化循环中所用的变量 (var i=0)
  2. 语句 1 是可选的，也就是说不使用语句 1 也可以。（比如在循环开始前已经设置了值时）
  3. 您可以在语句 1 中初始化任意（或者多个）值
    var i=2,len=cars.length;
    for (; i < len; i++) {
        document.write(cars[i] + "<br>");
    }

语句 2

  1. 通常语句 2 用于评估初始变量的条件。
  2. 语句 2 同样是可选的。
  3. 如果语句 2 返回 true，则循环再次开始，如果返回 false，则循环将结束。
如果您省略了语句 2，那么必须在循环内提供 break。否则循环就无法停下来。这样有可能令浏览器崩溃。

语句 3

  1. 通常语句 3 会增加初始变量的值。
  2. 语句 3 也是可选的
  3. 语句 3 有多种用法。增量可以是负数 (i--)，或者更大 (i=i+15)。
  4. 语句 3 也可以省略（比如当循环内部有相应的增量代码时）

for/in 循环

  1. JavaScript for/in 语句循环遍历对象的属性
    var person={firstName:"John",lastName:"Doe",age:25};
    var values = '';
    for (key in person) {
        values += person[key];
    }
