只要指定条件为 true，循环就可以一直执行代码。

while 循环

  1. While 循环会在指定条件为真时循环执行代码块。

while (条件) { 需要执行的代码 }

如果您忘记增加条件中所用变量的值，该循环永远不会结束。该可能导致浏览器崩溃。

do/while 循环

  1. do/while 循环是 while 循环的变体。
  2. 在检查条件是否为真之前，该循环会执行一次代码块，然后如果条件为真的话，就会重复这个循环。

do { 需要执行的代码 } while (条件);

别忘记增加条件中所用变量的值，否则循环永远不会结束！

比较 for 和 while

  1. 如果您已经阅读了前面那一章关于 for 循环的内容，您会发现 while 循环与 for 循环很像。

cars=["BMW","Volvo","Saab","Ford"]; var i=0; for (;cars[i];) { document.write(cars[i] + "<br>"); i++; } while (cars[i]) { document.write(cars[i] + "<br>"); i++; }
