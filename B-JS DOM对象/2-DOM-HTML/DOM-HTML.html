HTML DOM 允许 JavaScript 改变 HTML 元素的内容。

改变 HTML 输出流

  1. 在 JavaScript 中，document.write() 可用于直接向 HTML 输出流写内容。
    <!DOCTYPE html>
    <html>
        <body>

            <script>
                document.write(Date());
            </script>

        </body>
    </html>
绝不要使用在文档加载之后使用 document.write()。这会覆盖该文档。

改变 HTML 内容

  1. 修改 HTML 内容的最简单的方法是使用 innerHTML 属性。
    document.getElementById(id).innerHTML = 'new content';

改变 HTML 属性

  1. 如需改变 HTML 元素的属性，请使用这个语法：
    document.getElementById(id).attribute = new value;
